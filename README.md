## Portfolio Projects

### 🔍 [Healthcare Fraud in America: FCA Settlement Analysis](https://github.com/darbydawn/healthcare-fraud-fca-analysis)
Multi-phase fraud detection project using Python, SQL, and Tableau against public Medicaid (T-MSIS/DOGE), Medicare (CMS), DOJ FCA settlement, and OIG exclusion datasets.
- Phase 1 ✅ — Medicaid anomaly detection: 1,109 flagged records, 194 providers, 2 OIG submissions
- Phase 2 🔄 — Medicare provider utilization analysis (in progress)
- Phase 3 🔄 — DOJ FCA settlement cross-reference
- Phase 4 🔄 — OIG exclusion list matching
- Phase 5 🔄 — Tableau dashboard

### 📊 [Donor Analytics Dashboard — Nonprofit](https://github.com/darbydawn/donor-analytics-nonprofit)
Tableau analysis of 1,200+ donor records across donor type, geographic distribution, and retention rates.

### 🏥 [Population Health Cost Concentration](https://github.com/darbydawn/population-health-cost-concentration)
SQL and Power BI analysis of 116,000-member CMS synthetic dataset identifying cardiometabolic cost drivers and modeling $500K–$700K in intervention savings.

### 🤖 [Clinical AI Monitoring Framework](https://github.com/darbydawn/clinical-ai-monitoring-framework)
Power BI dashboard and framework for monitoring clinician engagement with AI-generated alerts — intervention rates, override patterns, and high-risk alert fatigue detection.

### 📋 [Epic Caboodle Data Quality Playbook](https://github.com/darbydawn/epic-caboodle-data-quality-playbook)
Production-grade data quality framework for Epic Caboodle analytics environments covering grain management, bridge tables, and slowly changing dimensions.

### ✅ [Clinical Data Quality Validation Playbook](https://github.com/darbydawn/clinical-data-quality-validation-playbook)
SQL-based validation framework for clinical datasets including orphan record detection, timeline anomaly identification, and unsigned note tracking.

### ⏱️ [Clinician Time Transparency Dashboard](https://github.com/darbydawn/clinician-time-transparency-dashboard)
Power BI dashboard tracking clinician time-in-chart across CCM and BHI programs with distribution analysis and open draft tracking.
